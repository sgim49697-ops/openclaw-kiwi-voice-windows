# OpenClaw Kiwi Voice Windows

Windows PC에서 Kiwi Voice, OpenClaw Gateway, Windows Node, 브라우저 자동화, VS Code/Codex wrapper를 안전하게 연결하기 위한 계획 문서 저장소다.

## 문서

- `docs/plans/`: 구현 계획과 버전별 로드맵
- `docs/policies/`: 운영 정책과 라이선스 준수 기준

## 라이선스

외부 자료를 추가할 때는 [라이선스 준수 정책](docs/policies/license-compliance.md)을 따른다.
